MyDay API

MyDay API is a simple Django REST Framework project designed to help me practice building CRUD-based backend systems. The API focuses on managing basic to-do lists and tasks, making it a lightweight foundation for future web or mobile apps.

This project serves as a learning tool to strengthen my understanding of Django, RESTful design, serializers, view classes, and handling HTTP requests.

Features

Create, read, update, and delete to-do list items

Clean and simple REST endpoints

Built using Django and Django REST Framework

Easy foundation for connecting to a frontend app
